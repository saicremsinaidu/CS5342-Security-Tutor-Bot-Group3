Local Network-Security Tutor & Quiz Bot

📌 Project Overview

This project is developed as part of CS5342 Network Security (Fall 2025).
Our system is a privacy-preserving intelligent tutor and quiz bot that:
	•	Answers course-related questions with citations.
	•	Generates quizzes (MCQs, True/False, Open-ended).
	•	Grades responses and provides feedback.
	•	Ensures all data remains local (no external servers).
